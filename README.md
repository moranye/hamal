# Hamal

NZ's boilerplate generator based on Command-line interface for front-end.

## Install

```shell
npm install -g hamal
```

## How to use?

### create and initialize a project

```shell
hamal init <project-name>
```

### list available templates

```shell
hamal list
```

### output help

```shell
hamal --help
```
